# ResumeAI - AI Resume Screener & Career Toolkit

ResumeAI is a full-stack AI-powered career assistant that helps job seekers analyze resumes, compare multiple job descriptions, generate cover letters, improve resume bullets, and prepare for interviews using Google's Gemini AI.

## Features

### ATS Resume Analysis

* Resume vs Job Description matching
* ATS compatibility score
* Missing keywords detection
* Matched skills identification
* Recruiter-style summary
* Section-wise scoring

### Multi-JD Comparison

* Compare a resume against multiple job descriptions
* Rank roles by suitability
* Identify skill gaps for each role

### AI Resume Bullet Rewriter

* Improve resume bullet points
* Stronger action verbs
* Better ATS optimization
* JD-specific tailoring

### AI Cover Letter Generator

* Formal tone
* Conversational tone
* Startup/Casual tone
* Tailored to resume and job description

### Interview Preparation

* Generate role-specific interview questions
* Technical questions
* Behavioral questions
* Resume-based questions
